# Product Studio v2 release notes

- Product-first customer storefront
- Manual and supplier products share one schema
- Per-color front, back and swatch images
- Secure product image uploads to Supabase Storage
- Front-only, back-only and front+back modes
- Independent artwork uploads and placement per side
- Independent front/back production previews
- Product-level decoration methods
- Product-level minimum quantities
- Side-based price surcharges
- Configurable front/back print areas
- Order detail shows both side files
- Encrypted multi-provider Integration Center
- Supplier fulfillment remains compatible with imported SKU data
